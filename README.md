# 🚀 Smart Habit Engine

> **“We don’t just track habits… we understand and evolve human behavior.”**

Smart Habit Engine is an AI-powered behavioral intelligence system that goes beyond traditional habit tracking.
It predicts your future, adapts to your emotions, and analyzes your behavioral patterns to help you build consistency intelligently.

---

## 🧠 Core Features

### 🔮 Future Self Simulation Engine

* Predicts your future based on current habits
* Shows dual timelines:

  * ✅ With habits
  * ❌ Without habits
* Calculates growth in:

  * Discipline
  * Focus
  * Energy

---

### 🧠 Emotion-Aware Habit Engine

* Adapts habits based on your mood
* 😞 Low mood → reduces workload
* 😃 High energy → increases challenge
* Makes habit building flexible and human-friendly

---

### 🧬 Habit DNA Engine

* Builds your behavioral DNA profile
* Detects:

  * Success/failure patterns
  * Peak productivity time
  * Mood impact on habits
* Example insight:

  > “You perform better at night”

---

## ⚙️ Tech Stack

### 💻 Backend

* Java (Spring Boot)
* REST APIs
* JPA / Hibernate

### 🎨 Frontend

* React
* Tailwind CSS
* Framer Motion

### 🤖 AI Integration

* OpenAI API (for insights & predictions)

---

## 📊 How It Works

1. User adds habits
2. Logs daily progress + mood
3. System analyzes behavior
4. AI generates insights & predictions
5. UI visualizes future & patterns

## 🚀 Getting Started

### 🔹 Clone the repository

```bash
git clone https://github.com/your-username/smart-habit-engine.git
cd smart-habit-engine
```

---

### 🔹 Backend Setup (Spring Boot)

```bash
cd backend
mvn spring-boot:run
```

---

### 🔹 Frontend Setup (React)

```bash
cd frontend
npm install
npm run dev
```
